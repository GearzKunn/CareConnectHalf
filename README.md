# Care Connect 

A comprehensive management platform designed to streamline support systems for NGOs, caretakers, and vulnerable populations. This application provides a centralized dashboard for tracking care requests, managing orphanages, and coordinating community support.

## 🚀 Features

* **Role-Based Dashboards**: Tailored views for Admins, NGOs, Caretakers, and Individual Users.
* **Request Management**: Real-time tracking and fulfillment of care and orphan support requests.
* **Secure Authentication**: Robust user verification and protected route management.
* **Responsive Design**: Fully optimized for both desktop and mobile operations.

## 🛠️ Tech Stack

* **Frontend**: React.js with TypeScript
* **Styling**: Tailwind CSS
* **State Management**: TanStack Query (React Query)
* **Routing**: React Router DOM
* **Build Tool**: Vite
